# Declutter Web

Next.js storefront, seller dashboard, admin dashboard, checkout, and order chat UI.

```bash
npm install
cp .env.example .env.local
npm run dev
```

Routes:

- `/` default Emox-inspired marketplace storefront
- `/themes/belo-fur`
- `/themes/lefore`
- `/themes/emox`
- `/dashboard`
- `/orders/demo`
